1. Installing Angular Language Extension for VS Code
The Angular Language Service extension provides better support for Angular development in Visual Studio Code.

Benefits
Auto-completion

Error detection

Go to Definition

Hover information

Template IntelliSense

Better debugging support

Installation Steps
Open Visual Studio Code.

Click Extensions (or press Ctrl + Shift + X).

Search for:

Angular Language Service
Install the extension published by Angular.

Restart VS Code if prompted.

Features
HTML template auto-completion

TypeScript IntelliSense

Error highlighting

Component navigation

Directive suggestions

3. launch.json Configuration
Example

{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "chrome",
            "request": "launch",
            "name": "Launch Angular",
            "url": "http://localhost:4200",
            "webRoot": "${workspaceFolder}"
        }
    ]
}
Explanation
Property	Purpose
version	Debug configuration version
type	Browser to launch
request	Launch or attach
name	Configuration name
url	Angular application URL
webRoot	Project root directory

5. Setting Breakpoints in TypeScript
Breakpoints pause execution at a specific line.

Example

export class StudentComponent {

    name = "Rahul";

    show()
    {
        console.log(this.name);
    }

}
Click beside

console.log(this.name);
A red dot appears.

When execution reaches this line, it pauses automatically.

6. Watches
A Watch lets you continuously monitor the value of a variable or expression while debugging.

Example

let total = price * quantity;
Add

total
to the Watch panel.

Whenever execution pauses, VS Code displays the current value.

Multiple Watches
price

quantity

total

student.name
Useful for tracking changing values.

10. Debugging Angular Services
Angular services usually contain business logic and API calls.

Example

@Injectable({
    providedIn: 'root'
})
export class StudentService {

    getStudents()
    {
        return this.http.get("/api/student");
    }

}
Set a breakpoint on

return this.http.get("/api/student");
When the component calls the service, execution pauses, allowing you to inspect:

Request URL

Parameters

Returned data
