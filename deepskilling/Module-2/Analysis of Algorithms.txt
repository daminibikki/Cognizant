Analysis of Algorithms (Simple Notes)
What is an Algorithm?

An algorithm is a step-by-step method to solve a problem.

Example

Find the largest number in an array.

Input:
[3, 8, 2, 10, 5]

Output:
10

Algorithm:

Assume first element is largest.
Compare it with every other element.
Update largest if needed.
Print largest.
What is Analysis of Algorithms?

Analysis means checking how good an algorithm is.

We mainly check:

Time (How long it takes)
Space (How much memory it uses)

Example:

Suppose there are two programs.

Program A takes 2 seconds

Program B takes 20 seconds

Program A is better.

But what if input becomes huge?

That's why we analyze algorithms.

Factors That Affect Running Time

Running time depends on

Input size
Computer speed
Programming language
Algorithm used

When comparing algorithms, we ignore hardware and language.

We only compare the algorithm.

Time Complexity

Time complexity tells

How the running time increases when input size increases.

We don't count seconds.

We count operations.

Example

"Asymptotic" means

How an algorithm behaves for very large input.

Example
