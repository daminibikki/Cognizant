What is a Linked List?

A Linked List is a linear data structure where elements (called nodes) are connected using links (references/pointers).

Unlike an array, the elements are not stored in contiguous memory.

Each node contains:

Data – the value stored.
Next – reference to the next node.

Example:

Head
 ↓
+------+-------+    +------+-------+    +------+------+
| 10   |   •---|--->| 20   |   •---|--->| 30   | NULL |
+------+-------+    +------+-------+    +------+------+
Why Do We Need Linked Lists?

Suppose you have an array:

10 20 30 40

If you want to insert 25 between 20 and 30, all elements after 20 must be shifted.

This takes O(n) time.

In a linked list:

10 → 20 → 30 → 40

Just change one link:

10 → 20 → 25 → 30 → 40

No shifting is required.

Node Structure in Java
class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

Types of Linked Lists
1. Singly Linked List

Each node stores only the next node.

10 → 20 → 30 → NULL

Most commonly used.

2. Doubly Linked List

Each node stores:

Previous node
Next node
NULL ← 10 ⇄ 20 ⇄ 30 → NULL

Can move forward and backward.

3. Circular Linked List

Last node points back to the first node.

10 → 20 → 30
↑           ↓
└───────────┘

No NULL at the end.

4. Circular Doubly Linked List

Each node has:

Previous
Next

First and last nodes are connected.

Advantages of Linked List
Dynamic size (can grow or shrink).
Fast insertion at the beginning.
Fast deletion at the beginning.
No need for contiguous memory.
Efficient memory usage when size changes frequently.
Disadvantages of Linked List
No direct access by index.
Extra memory is needed for storing links.
Searching is slower than arrays.
More complex implementation than arrays.
Array vs Linked List
Array
Stored in contiguous memory.
Fixed size.
Direct access using index.
Access time: O(1).
Insertion/Deletion: O(n).
Linked List
Stored in non-contiguous memory.
Dynamic size.
No direct index access.
Access time: O(n).
Insertion at beginning: O(1).
Java LinkedList Class

Java provides a built-in LinkedList class in the java.util package.

import java.util.LinkedList;

public class Main {
    public static void main(String[] args) {

        LinkedList<Integer> list = new LinkedList<>();

        list.add(10);
        list.add(20);
        list.add(30);

        System.out.println(list);
    }
}

Output

[10, 20, 30]

Common Interview Questions
Reverse a Linked List
1 → 2 → 3

↓

3 → 2 → 1
Find Middle Node

Use Slow and Fast Pointers.

Slow moves one step.
Fast moves two steps.
When fast reaches the end, slow is at the middle.

Time: O(n).

Detect a Cycle

Use Floyd's Cycle Detection Algorithm (Tortoise and Hare).

Slow moves one step.
Fast moves two steps.
If they meet, a cycle exists.

Time: O(n).
